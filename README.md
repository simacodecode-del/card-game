A simple, responsive Memory / Concentration card game built with plain HTML, CSS and JavaScript. This project is intended as a lightweight starter you can extend (themes, levels, multiplayer, scoreboard).

Features:
- 4x4 grid (8 pairs) using emoji cards (no assets needed)
- Flip animation and match detection
- Move counter and timer
- Restart and win overlay

How to run:
1. Clone or download this repository.
2. Open `index.html` in your web browser (no server required).
3. Play, press Restart to reshuffle.

Possible next steps:
- Add multiple difficulty levels (4x4, 6x6)
- Add persistent high score using localStorage
- Replace emoji with card image assets / theme packs
- Add keyboard accessibility and improved ARIA labels
